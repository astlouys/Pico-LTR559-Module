# Pico-LTR559-Module
Add-on C-Language module to merge with your existing Raspberry Pi Pico (C-Language) project / program,
to support a LTR559 sensor, using I2C communication protocol.

The LTR559 sensor can read the ambient light level and it can also be used as a (very close) proximity sensor.

You must not simply clone the repository to take advantage of this module. Please
read the User Guide and carefully follow the given instructions.

Enjoy !
